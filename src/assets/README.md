# Assets - Custom Icons

Place your custom icon images in this folder with the following names:

## Navigation Icons (for sidebar menu)
- **timesheet-icon.png** - Icon for "Учет времени" (Time Tracking)
- **projects-icon.png** - Icon for "Проекты" (Projects)
- **employees-icon.png** - Icon for "Сотрудники" (Employees)

## Header Icons (for page titles)
- **timesheet-header-icon.png** - Icon for TimeSheet page title (optional)

## Recommended Icon Specs
- **Size:** 24x24px for navigation icons, 48x48px for header icons
- **Format:** PNG with transparency (recommended)
- **Style:** Flat design, monochromatic or colored

## How It Works
The app will automatically load images from this folder if they exist. If images are not found, it will fall back to emoji icons (⏱️, 📊, 👥).

Simply add your PNG files to this folder and they will appear in the app without any code changes needed.
